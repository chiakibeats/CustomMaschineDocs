How To Map Parameters To Knobs And Buttons​

  1. Press CHANNEL button to switch to custom MIDI map mode
  2. Navigate to the page you want to use
  3. Enter MIDI mapping mode from Ableton
  4. Click parameter and turn knob or push button to learn
Limitation on Learning

Due to technical limitation, the script sends knob touch sensor MIDI messages. This means touch sensor messages could overwrite your MIDI mapping.

To map parameters correctly, please follow these steps on learning.

  1. Click target parameter
  2. Turn knob that you want to map and hold your hand to the knob
  3. Click blank area of MIDI mapping window to finish learning
  4. Release your hand from knob

We will try to resolve this problem in future version!
